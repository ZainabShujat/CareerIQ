# CareerIQ - Smart Career Recommender System

A full-stack AI-driven career recommendation platform designed specifically for Indian students and professionals.

## 🚀 Overview

CareerIQ helps users discover careers that align with their personality, skills, and aspirations through:
- 7-minute psychometric assessment
- AI-powered career matching
- Real Indian market insights
- Skill validation tests
- Live industry trends

## 🎯 Key Features

- ✅ AI-based career recommendation
- ✅ 24-question personality assessment
- ✅ Analysis of 10 personality traits
- ✅ 7 happiness dimensions evaluation
- ✅ Real Indian salary & market data
- ✅ Skill validation tests
- ✅ Career insights search
- ✅ Sector-specific guidance
- ✅ Mobile-responsive design
- 🔜 AI Career Chatbot (Coming Soon)

## ⚙️ Tech Stack

### Frontend
- React
- React Router
- TailwindCSS

### Backend
- Netlify Serverless Functions
- MongoDB Atlas
- Custom AI Algorithm

### Deployment
- Frontend & API: Netlify
- Database: MongoDB Atlas
- CI/CD: GitHub

## 🚀 Getting Started

### Prerequisites
- Node.js >= 14.x
- npm >= 6.x

### Installation
```bash
# Clone the repository
git clone https://github.com/yourusername/careeriq.git

# Install dependencies
cd careeriq
npm install

# Start development server
npm start
```
